期末專題

HTML5 水果忍者 (Fruit Ninja)

這是一個以原生 HTML5 Canvas 與 JavaScript 物理動畫開發的「水果忍者」網頁版遊戲。玩家可以用滑鼠或手指滑動，在螢幕上切碎各式水果並避免切到炸彈！

遊戲規則與操作：

  • 切水果： 按住滑鼠左鍵並拖曳（或在觸控螢幕上滑動）劃出劍氣切碎水果。
  • 生命值： 漏掉水果會扣 1 顆心，切到炸彈直接扣除所有生命（或扣 1 顆心）。共有 3 顆心。
  • 連擊 (Combo)： 同一次劃動切碎 3 個或更多水果可獲得額外 Combo 加分！
  • 登入存檔： 登入後遊戲結束會自動將你的高分上傳至排行榜！
